# MICモーグルホームページ リニューアルプロジェクト

現行サイト（mogul-mic.com）を Apple 風のデザインに刷新するプロジェクトです。
試作環境は GitHub Pages（vanwhist/mic-site-mockup）、公開予定ドメインは `mogul-mic.site`。
レンタルサーバーは使わず、GitHub Pages 単体で運用します。

## 返答スタイル
- 返答は**日本語**で、**簡潔**に。

## 実装方針
- **トップページ**：フルスクラッチで実装する。
- **コーチ紹介・選手紹介・お知らせ**：データ連動で実装する（手書きの静的HTMLにしない）。

## 作業ルール
- **ファイルの削除は必ず事前に確認を取ること。** 上書きも同様に、内容を確認してから行う。

## 主なファイル
- トップページ本体：`prototype-site/index.html`
- 引き継ぎメモ（最新）：`handoff_MICホームページリニューアル_20260723.md`
- 現在のタスク：`タスク.md`
- 作業ログ：`作業ログ.md`
